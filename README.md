# Quick Notes

> A simple notes website, support large contents with IndexedDB

## Development

- Use `yarn install` or `npm install` to install node_modules
- Run `yarn dev`
- Go to http://127.0.0.1:8081

## Build

- Run `yarn build`

## Start the website

- Run `yarn start`
